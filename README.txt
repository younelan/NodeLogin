node zone
---------
Simple node.js demo with mongodb, express and a theme engine. Shows how to create a simple member zone, storing the users in mongo db. This includes a very simple theme engine

to use:
_ make sure node.js is installed and in your executable path
_ make sure mongodb is installed and running
_ change to nodezone directory
_ make sure there is nothing running on port 80 (or change port)
_ run "node app"
